# Table Designer

Interactive 3D table designer with cut list and build plan for a Grade 9
woodworking project.

**Live version:** https://yourusername.github.io/table-designer/

## What it does

- Edit table dimensions in the browser
- View assembled and exploded 3D models with measurements
- Generate a cut list automatically
- Export the cut list as a CSV file
- Print the whole plan (drawings + cut list + build steps)

## How to build the table

The table uses softwood (pine or SPF) and is assembled with glue and screws.
No specialised tools needed — just a mitre saw, drill, and screwdriver.
The build plan is included in the app.
